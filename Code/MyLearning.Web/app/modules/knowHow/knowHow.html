How I did it?

Steps to follow :

  1. Setup Web project - create empty Website. Create folder structure.

  2. Setup Web API project - add Web API 2 Empty project.

  3. Setup external reference through Bower - add Bower file. Add .bowerrc which is used to set the target location for referenced libraries.

  4. Add Bootstrap - add bootstrap in Bower.json. Add bootstrap css and js files in index.html. Add corresponding classes in html files. Ref http://getbootstrap.com/components/

  5. Add routing - add angular-route. Add individual routes in app.js

  6. Add custom.less - add less in bower.json and add reference to js file in index.html

  7. Enable CORS Add package for WebApi.CORS and EnableCorsAttributefor Cross Origin Resource Sharing in WebApiConfig by defining url of website.

  8. JSON formatting - set JsonFormatterContractResolver as CamelCasePropertyNamesContractResolver

  9. Remove # from url - set base href and set $locationProvider.html5Mode(true) in app.js

  10. Resolve page refresh issue - add web.config in Web site project and set rules and conditions to rewrite url.

  11. Flexible nav and page header - create MODULE_NAMES constant in app.js and use that for rendering navs and page header.

  12. Gulp implemented for continuous build and using minified files during run.

  13. Notification Service implemented to display andy warning or success message below header.

  14. Exception handler implemented to catch almost any type of exception using exceptionHandler service by Angular.

  15. HttpInterceptor implemented to catch all types of response from Web API.
